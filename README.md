# AI Web Application Learning Assistant

An AI-powered assistant that helps users learn how to use any website or application step-by-step.

## Features

- Analyze any website
- Generate step-by-step guides
- Beginner-friendly instructions
- Works offline with local AI
- Modern UI

## How It Works

1. Enter a website URL
2. AI analyzes the website
3. Enter the task you want to perform
4. AI generates instructions

## Tech Stack

- Python (Flask)
- HTML/CSS/JavaScript
- Ollama (TinyLlama)
- Local AI models
