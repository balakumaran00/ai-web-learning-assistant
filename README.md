# AI Web Learning Assistant

An AI-powered assistant that helps users learn how to use any website or application step-by-step.

## Features

- Analyze any website
- AI-generated instructions
- Step-by-step guidance
- Beginner-friendly
- Fast responses
- Clean UI

## Tech Stack

- Python (Flask)
- HTML/CSS/JavaScript
- Ollama (TinyLlama)
- Local AI Models

## Installation

Clone the repository:

git clone https://github.com/balakumaran00/ai-web-learning-assistant.git

Install dependencies:

pip install flask requests beautifulsoup4

Run the app:

python app.py

Open browser:

http://127.0.0.1:5000
